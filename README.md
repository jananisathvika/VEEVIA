## ✨ What is Veevia?

Veevia is an AI-powered interior design platform that transforms any house into a beautiful, personalised home.

- 📸 Upload a photo of your room
- 🤖 Get instant AI design suggestions
- 🎨 Design it yourself using our DIY studio
- 👩‍🎨 Connect with a verified professional designer
- 🔨 Get it built end-to-end by trusted contractors

---

## 🚀 Features

| Feature | Description |
|---|---|
| 🤖 AI Room Analysis | Upload any room photo and get instant personalised suggestions |
| 🎨 DIY Design Studio | Drag and drop furniture, colours and décor elements |
| 👩‍🎨 Verified Designer Network | Connect with professional designers at affordable prices |
| 🔨 End-to-End Execution | From design to delivery — contractors, vendors and painters |
